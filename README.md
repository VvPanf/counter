# Counter 

Простой список счётчиков на Android

## Возможности

- Просмотр списка счётчиков на главном экране
- Выбор счётчика
- Инкрементирование и декрементирование счётчиков
- Изменение название и цвета счётчика
- Удаление счётчика

## Стек технологий

- Kotlin 1.9
- Android
- Gradle
- Koin
- Room
- MVVM
